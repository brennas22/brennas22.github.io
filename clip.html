Clip

Clip

In Interaction Design I we were tasked with creating an interface for desktop and mobile that helped a user keep track of some sort of data. A problem that struck very close to home for me because of being a transfer student was the college search process. This is a big problem for thousands of teens who have a number of desires for prospective schools. While there are many websites that allow students to search by criteria, there isn't a way to save information from various sources all in one place; a student is considering not only the raw stats of the school, but also news, pictures and student testimonials all of which are found in many places on the web.

this video is a walkthrough of the desktop version of Clip.

Features

Clip allows users to filter schools based on a number of criteria. Once they've found a school they're potentially interested in they can create a collection centered around that school. Then, a plug in (both mobile and desktop) allows them to save external content to the school's collection, which they can then go back into Clip and organize however they choose.

this video is a walkthrough of the mobile version of Clip.
