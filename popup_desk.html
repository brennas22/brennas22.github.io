Features

Clip allows users to filter schools based on a number of criteria. Once they've found a school they're interested in they can create a collection centered around that school. Then, a plug-in (both mobile and desktop) allows them to save external content to the school's collection, which they can then go back to in Clip and organize however they choose.

Recent trends in web technology have shown a large uptick and continued growth in the number of mobile users, especially when it comes to browsing content.

In the mobile version of Clip the user can view their predetermined choices as well as see school matches, make and edit collections and, perhaps most importantly for mobile, add external items. The mobile version of Clip makes it extremely easy for users to save any content they come across directly to the relevant collection, where they can then view it on desktop as well.

Process Book

Fall 2017

Problem Statement

How might we create an efficient, productive and portable space that is adaptable for users between the ages 14 to 22 years old to do schoolwork on and off campus?

Competitive Product Research / IP Search

My first design attempt was built into the existing form at the time. There needed to be a way to input different pieces of data for one item (it later turned out this was all linked in the backend).

My initial thought was to just enter each piece of data, one at a time, and they would appear as a line in the form.

This design was flawed though because there was no way of knowing what the next information you needed to know would be, and what if you didn't know a portion? How would you leave it blank and go to the next section? It is also very unclear which element has focus.

Product: 3-in-1 Product: A desk, chair, bag (Convinenture)

Stengths: Increased Productivity, Portable and Lightweight, Compact, Multipurpose kit, Carters to various lifestyle, Water Resistant SB EVA Material, Slots for cord management, Laptop ventilation holes

Weaknesses: Size limitations: 17 in, Only support under 230 lbs, Not a reasonable price: $193, Not stylish, No Portable charging system

Key Takeaways

This project was the first that I completed end to end, from problem identification all the way through a complete design. I learned a lot about interacting with users and the design thinking process as well as some technical skills such as Balsamiq, Adobe XD, InVision and user testing. I was also able to strengthen preexisting skills such as sketching and taking feedback in order to effectively iterate. If I were to change an aspect of my process or final product I would do more extensive user testing after my first round of visual design because I made a significant number of organizational changes.
